# Contributing to Keelson

Thank you for your interest in contributing to Keelson! Keelson is an agentic framework designed to bring structured workflows to AI coding assistants like Claude Code and Codex.

## How to Contribute

1. Fork the repository and create your branch from `main`.
2. Ensure you have Node.js 18 or newer installed.
3. Run `npm ci` to install dependencies.
4. Make your changes in the appropriate directory (e.g., `skills/core/` for skill modifications).
5. Ensure `npm run lint` and `npm run test` pass.
6. Create a pull request.

## Code of Conduct
Please follow our [Code of Conduct](CODE_OF_CONDUCT.md) in all your interactions with the project.
