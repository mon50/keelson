# Reforge Status Examples

## Example 1: Ideation Phase
```text
Target Spec: default
Current Phase: ideation
Blockers: None
Unresolved Questions: 3 pending
Approval State: Not approved
Task Summary: Not planned
Last Verify State: None
Next Recommended Command: reforge-resume
```

## Example 2: Ambiguous Spec
```text
Target Spec: ambiguous
Error: Multiple specs found. Please specify the target spec.
Available specs: default, admin-panel
Next Recommended Command: reforge-status "spec-name"
```
