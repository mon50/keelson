# Security Policy

## Supported Versions
Currently, only the latest release of Keelson is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability
If you discover a security vulnerability in Keelson, please report it via GitHub Advisory or email the maintainers directly. Do not open a public issue for security vulnerabilities.
