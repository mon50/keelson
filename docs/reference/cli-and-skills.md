# CLI and Skills Reference

**Audience:** Daily users of Reforge.
**Prerequisites:** Reforge installed.
**Expected Outcome:** Know which command to use when.

## CLI
- `npx reforge install`: Installs skills to `.claude/skills` or `.agents/skills`.
- `npx reforge doctor`: Validates workspace.
- `npx reforge uninstall`: Removes Reforge.
