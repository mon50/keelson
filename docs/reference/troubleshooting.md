# Troubleshooting

- Missing workspace: run `/keel-requirements "<idea>"`.
- User story mismatch: revise with `/keel-us` or return to `/keel-requirements`.
- Existing implementation constraints missing: run `/keel-design`.
- Prototype does not match the intended experience: return to `/keel-us`.
- Prototype does not match implementation design: return to `/keel-design`.
- Plan is not approved: run `/keel-plan`.
- Implementation is blocked by missing product detail: return to the owning earlier phase instead of guessing.
