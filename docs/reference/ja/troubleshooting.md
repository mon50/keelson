# トラブルシューティング

<sub>[← Keelson ドキュメント](../../README.md) · [English](../troubleshooting.md) | 日本語</sub>

- ワークスペースがない: `/keel-requirements "<idea>"` を実行する。
- ユーザーストーリーの不一致: `/keel-us` で改訂するか `/keel-requirements` に戻る。
- 既存実装の制約が不足: `/keel-design` を実行する。
- プロトタイプが意図した体験と合わない: `/keel-us` に戻る。
- プロトタイプが実装デザインと合わない: `/keel-design` に戻る。
- プランが未承認: `/keel-plan` を実行する。
- プロダクトの詳細不足で実装がブロックされた: 推測せず、所有元の前段フェーズへ戻る。
