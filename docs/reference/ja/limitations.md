# 制限事項

<sub>[← Keelson ドキュメント](../../README.md) · [English](../limitations.md) | 日本語</sub>

- Keelson は requirements・user stories・design・prototype・タスク計画を収束させます。完全な自律アプリビルダーではありません。
- 簡易プロトタイプはレビューの根拠であり、本番 UI コードではありません。
- ブラウンフィールドのリポジトリ調査は機能スコープであり、根拠に基づきます。
- `/keel-impl` は承認済みプランのタスクを一度に 1 つ実装し、承認済みアーティファクトが不十分なときは停止すべきです。
