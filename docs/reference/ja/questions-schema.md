# 質問

<sub>[← Keelson ドキュメント](../../README.md) · [English](../questions-schema.md) | 日本語</sub>

未解決の疑問は、それを所有するアーティファクトの中に置きます。

- Requirements の疑問: `requirements.md`
- ユーザーストーリーと操作の疑問: `user-stories.md` または `us-mock.html`
- 実装・デザインの疑問: `design.md`
- プロトタイプレビューの疑問: 必要に応じて `prototype-notes.md`

未解決の疑問は、そのフェーズの承認をブロックします。
