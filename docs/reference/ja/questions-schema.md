- Requirements の疑問: `01-requirements/requirements.md`
- ユーザーストーリーと操作の疑問: `02-user-stories/user-stories.md` または `02-user-stories/us-mock.html`

- プロトタイプレビューの疑問: 必要に応じて `04-prototype/prototype-notes.md`

未解決の疑問は、そのフェーズの承認をブロックします。
