# プロトタイプアーティファクトのアーキテクチャ

<sub>[← Keelson ドキュメント](../../README.md) · [English](../ui-artifact-architecture.md) | 日本語</sub>

`prototype.html` は `/keel-proto` が作成する簡易レビュー用アーティファクトです。

次のアーティファクトに根ざしている必要があります。

- `requirements.md`
- `user-stories.md`
- `us-mock.html`
- `design.md`

プロトタイプはユーザーストーリーの体験とデザイン方針を検証します。本番コードではなく、実装の出典としての `design.md` を置き換えるものでもありません。
