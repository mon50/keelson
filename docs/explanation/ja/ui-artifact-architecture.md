# プロトタイプアーティファクトのアーキテクチャ

<sub>[← Keelson ドキュメント](../../README.md) · [English](../ui-artifact-architecture.md) | 日本語</sub>

`04-prototype/prototype.html` は `/keel-proto` が作成する簡易レビュー用アーティファクトです。

次のアーティファクトに根ざしている必要があります。

- `01-requirements/requirements.md`
- `02-user-stories/user-stories.md`
- `02-user-stories/us-mock.html`
- `03-design/design.md`

プロトタイプはユーザーストーリーの体験とデザイン方針を検証します。本番コードではなく、実装の出典としての `03-design/design.md` を置き換えるものでもありません。
