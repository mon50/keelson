# Hello Reforge Tutorial

**Audience:** Developers new to Reforge.
**Prerequisites:** Node.js 18+, Claude Code or Codex installed.
**Expected Outcome:** You will build a daily report prototype.

## Step 1: Install Reforge
```bash
npx reforge install
```

## Step 2: Initialize
In your AI agent:
```
/reforge-init "Daily Report App"
```

## Step 3: Resume
Answer questions to build the spec.
```
/reforge-resume
```

## Step 4: Render
```
/reforge-render
```

## Step 5: Plan & Implement
```
/reforge-plan
/reforge-impl
```
