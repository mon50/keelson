# Hello Keelson チュートリアル

<sub>[← Keelson ドキュメント](../../README.md) · [English](../hello-keelson.md) | 日本語</sub>

**対象読者:** Keelson をはじめて使う開発者。
**前提条件:** Node.js 18 以上、Claude Code または Codex がインストール済み。
**到達点:** 承認済みの Inception アーティファクトと、最初の実装プランを作成します。

## ステップ 1: Keelson をインストール
```bash
npx keelson-cli install
```

## ステップ 2: Requirements
AI エージェント内で実行します。
```
/keel-requirements "Daily Report App"
```

## ステップ 3: User Stories と Design
次のフェーズに進む前に、各アーティファクトを承認します。
```
/keel-us
/keel-design
```

## ステップ 4: Prototype
```
/keel-proto
```

## ステップ 5: Plan & Implement
```
/keel-plan
/keel-impl T-001
```
