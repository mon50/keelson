# Hello Keelson Tutorial

<sub>[← Keelson Docs](../README.md) · English | [日本語](ja/hello-keelson.md)</sub>

**Audience:** Developers new to Keelson.
**Prerequisites:** Node.js 18+, Claude Code or Codex installed.
**Expected Outcome:** You will create approved Inception artifacts and a first implementation plan.

## Step 1: Install Keelson
```bash
npx keelson-cli install
```

## Step 2: Requirements
In your AI agent:
```
/keel-requirements "Daily Report App"
```

## Step 3: User Stories And Design
Approve each artifact before moving to the next phase.
```
/keel-us
/keel-design
```

## Step 4: Prototype
```
/keel-proto
```

## Step 5: Plan & Implement
```
/keel-plan
/keel-impl T-001
```
