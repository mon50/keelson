# プロトタイプのレビュー

<sub>[← Keelson ドキュメント](../../README.md) · [English](../reviewing-prototypes.md) | 日本語</sub>

`/keel-proto` は、`requirements.md`・`user-stories.md`・`us-mock.html`・`design.md` が承認されたあとに使います。

`prototype.html` は次の 2 点をレビューします。

- ユーザーストーリーと US モックが記述する体験を提供しているか?
- `design.md` の実装方針と既存 UI の根拠に従っているか?

体験が誤っていれば `/keel-us` に戻ります。デザイン方針が誤っていれば `/keel-design` に戻ります。
